# Data Visualization and Storytelling

## Objective
This project focuses on exploring a dataset using visualization techniques and presenting meaningful insights through charts.

## Dataset Description
The dataset contains both numerical and categorical features including age, gender, department, experience, salary, city, and education.

## Tools Used
Python, Pandas, Matplotlib, Seaborn

## Visualizations Performed
Histogram for salary distribution, bar chart for department comparison, scatter plot for experience versus salary, and heatmap for correlation analysis.

## Key Insights
Salary increases with experience, IT and Finance have higher employee count, most salaries are in mid-range, and strong correlation exists between experience and salary.

## Outcome
The project successfully demonstrates how data visualization helps in understanding patterns and communicating insights effectively.
